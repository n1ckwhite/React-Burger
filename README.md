# Проект: Stellar Burger

Учебный проект факультета "React разработчик" Яндекс.Практикума.

В приложении реализованы такие фишки, как: 
- Перетаскивание бургеров, 
- Создание заказа, роутинг,
- Модальные окна, 
- Авторизация, 
- Регистрация, 
- Восстановление пароля, 
- Изменение профиля,
- Unit тесты,
- E2E тесты

## Технологии проекта
- HTML,
- flexbox верстка,
- переходы, 
- Css modules,
- React,
- React Router,
- React DnD,
- Redux,
- TypeScript,
- Web Socket,
- Jest,
- Cypress.
